# USDT Tracker PWA

A Progressive Web App for tracking USDT buy/sell trades with P&L statements.

## Files
- `index.html` — Main app
- `manifest.json` — PWA manifest
- `sw.js` — Service worker (offline support)
- `icons/` — App icons

## How to host FREE on GitHub Pages

1. Go to https://github.com and create a free account
2. Click **New repository** → name it `usdt-tracker` → Public → Create
3. Upload all these files (index.html, manifest.json, sw.js, icons folder)
4. Go to **Settings** → **Pages** → Source: `main` branch → Save
5. Your app URL will be: `https://YOUR-USERNAME.github.io/usdt-tracker`

## Install on iPhone

1. Open the URL above in **Safari** (must be Safari, not Chrome)
2. Tap the **Share** button (box with arrow at bottom)
3. Scroll down → tap **"Add to Home Screen"**
4. Tap **Add**
5. App icon appears on your home screen ✅

## Features
- Buy/Sell USDT recording with your own manual rates
- AED, USD, PKR currency support
- Inventory tracker
- P&L statement with charts
- Export to Excel (.xlsx) and PDF
- Works fully offline after install
- Dark mode support
